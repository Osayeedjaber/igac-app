# IGAC App URLs map

Here are the available routes in the IGAC app, automatically derived from the `src/app` directory:

## Public & Main Pages
- `/` - Main Landing Page
- `/about` - About Page
- `/contact` - Contact Page
- `/events` - Events Page
- `/join` - General Join/Recruitment Page
- `/register` - General Registration Page
- `/team` - Team Directory Page
- `/team/[slug]` - Individual Team Member Profile Page
- `/maintenance` - Maintenance Mode Page
- `/regions/ctg` - Chattogram Region Page

## IMUN Section (Imperial Model United Nations)
- `/imun` - Base IMUN Landing Page
- `/imun/about` - About IMUN
- `/imun/committees` - IMUN Committees
- `/imun/schedule` - IMUN Schedule
- `/imun/secretariat` - IMUN Secretariat
- `/imun/register` - IMUN Base Registration Hub
- `/imun/register/ca` - IMUN Campus Ambassador Registration
- `/imun/register/del` - IMUN Delegate Registration
- `/imun/register/eb` - IMUN Executive Board Registration

## Admin & CRM Portals
- `/admin` - Main Admin Redirect / Auth
- `/admin/dashboard` - Secure Full Admin Dashboard (for Site Settings, Team, Events, IMUN Config)
- `/portal/login` - General Portal Login
- `/portal/dashboard` - Secure General Portal Dashboard
- `/portal/crm/login` - CRM Portal Login
- `/portal/crm` - Secure CRM Internal Portal
- `/portal/scan` - Portal Scanning Utility
